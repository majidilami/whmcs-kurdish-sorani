# WHMCS Kurdish Sorani Language Pack

Kurdish Sorani language files for WHMCS admin area and client area, including a Kurdistan flag icon for language menus.

## Author

Majid Mozafari  
Website: https://majidilami.com/

## Documentation

How to add Kurdish language to WHMCS:  
https://majidilami.com/how-to-add-kurdish-language-to-whmcs/

## Included Files

- Admin area translation
- Client area translation
- Kurdistan flag icon

## Folder Structure

- `admin/kurdish-sorani.php`
- `client/kurdish-sorani.php`
- `flag/kurdish-sorani.svg`

## Installation

### Client Area Language

Upload:

`client/kurdish-sorani.php`

To:

`/lang/kurdish-sorani.php`

### Admin Area Language

Upload:

`admin/kurdish-sorani.php`

To:

`/admin/lang/kurdish-sorani.php`

### Flag Icon

Upload:

`flag/kurdish-sorani.svg`

To your WHMCS flag image location, for example:

`/templates/your-template/img/flags/kurdish-sorani.svg`

If your WHMCS theme supports custom language flags, add the needed CSS or template code to display the Kurdistan flag in the language menu.
